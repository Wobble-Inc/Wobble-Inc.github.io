Desktop software

Gestione dati aziendali

FlimBox è un’app desktop pensata per creare, organizzare e gestire tabelle personalizzate in contesto locale, con un approccio semplice e professionale.

Update flow

Controllo versione centralizzato

L’app controlla la disponibilità di nuove release online e utilizza update.json per leggere versione, note di rilascio e link di download.

Distribution

GitHub Releases

I pacchetti di aggiornamento vengono distribuiti tramite GitHub Releases, così da mantenere il processo trasparente, affidabile e facilmente verificabile.

Aggiornamenti automatici

FlimBox può verificare online la presenza di nuove versioni e guidare l’utente verso l’installazione del setup più recente in modo lineare.

02

Download sicuro con SHA-256

I file distribuiti possono essere verificati tramite hash SHA-256, offrendo maggiore integrità nel processo di aggiornamento dell’app desktop.

03

Configurazione tramite update.json

Il file update.json centralizza le informazioni essenziali: versione disponibile, note di rilascio e collegamenti ai download ufficiali.
